# Computer Setup

## Requirements 
1. x86 architecture 
2. `>=`50GB disk space
3. `>=` 16GB RAM
4. Wi-Fi / Wi-Fi Adapter for communication
5. AC Power Recovery (recommended)

## BIOS Configuration
1. Enable AC Power Recovery in the settings to `On` (if possible)
2. Enable RTC Alarm (if possible)
    - set to 0:00 daily
